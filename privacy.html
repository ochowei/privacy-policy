Privacy Policy

This API service does not collect, store, or share any personal information from users.

The service only fetches publicly available content from Facebook Pages using the Facebook Graph API with the approved Page Public Content Access permission. This content is used solely for content aggregation and analysis purposes.

No Facebook login, account data, or private user information is involved at any point in the process.

We are committed to responsible and transparent data handling practices. If you have any questions, please feel free to contact us.
